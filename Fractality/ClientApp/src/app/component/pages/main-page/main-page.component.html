Множество Мандельброта

Множество Мандельброта - это название для геометрической фигуры, которая является результатом математического процесса, открытого Бенуа Мандельбротом в 1970-х годах. Множество Мандельброта является примером фрактала и представляет собой геометрический объект, который характеризуется самоподобием, то есть он содержит в себе уменьшенные копии самого себя. Чтобы понять, что такое множество Мандельброта, нужно сначала разобраться в процессе, лежащем в основе его создания. Этот процесс включает в себя итерацию двух математических функций: z -> z^2 + c где z - комплексное число (z = x + iy), а c - любое другое комплексное число. Итерация начинается с заданного значения z, и на каждой итерации к z применяется функция z^2 + c.

Множество Джулия

Множество Джулии - это ещё один тип фрактала, который был открыт в 1918 году математиком Гарольдом Фрайлендером Джулия. Это множество названо в честь своего создателя. Множество Джулии, как и множество Мандельброта, является результатом итерации функции: f(z) = z^2 + c, где z — комплексное число, а c — некоторое комплексное число. Однако, в отличие от множества Мандельброта, множество Джулии состоит из точек, которые “расплываются” при итерациях. То есть, если мы начнём итерировать функцию f(z) с некоторым начальным значением z0, то после нескольких итераций точка z будет стремиться к бесконечности. Множество Джулии имеет очень интересный и сложный рисунок, который зависит от значения параметра c. Например, если c = 0, то множество Джулии представляет собой отрезок прямой между точками -1 и +1. Если же c близко к 0, но не равно 0, то мы получаем множество точек, которые образуют красивый и сложный узор.
